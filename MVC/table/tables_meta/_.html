Meta data Types

Meta data is maintained by the sytem and is readonly. all Meta data field names start with _

  1. Primary Key

    _pk Is an integer and auto incrementate as new records are added. Current name is pk,

  2. _date_created

    Date record was created, the data is in the change log, but no code written to access it

  3. _date_modified

    Date record was modified, the data is in the change log, but no code written to access it

  4. _date_deleted

    Date record was deleted, the data is in the change log, but no code written to access it
